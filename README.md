# (RU)
## Название проекта: 

Веб-приложение для сотрудников склада интернет-магазина носков

## Описание проекта:

Веб-приложение, с помощью которого склад может учитывать и автоматизировать учет товаров на складе интернет-магазина носков

## Стек технологий

  - Java 17
  - Maven
  - Spring Boot
  - Spring Web
  - GIT
  - REST
  - Swagger
  - Lombok
